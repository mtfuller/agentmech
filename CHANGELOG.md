# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added
- CI/CD pipeline for automated testing and NPM publishing
- GitHub Actions workflows for continuous integration
- Automated NPM publishing on release creation
- Version check workflow for pull requests
- Comprehensive publishing documentation
- LICENSE file (ISC)
- `.npmignore` for controlling package contents

### Changed
- Updated README with NPM installation instructions
- Enhanced package.json with repository, homepage, bugs, and files fields
- Added additional keywords for better NPM discoverability

## [1.0.0] - Initial Release

### Added
- AI-powered workflow generation from natural language
- Web UI for browsing and managing workflows
- Ollama integration for local AI model execution
- Real-time streaming support for LLM responses
- Multimodal support for images and documents
- MCP (Model Context Protocol) integration
- RAG (Retrieval-Augmented Generation) support
- Workflow tracing and logging capabilities
- Automated test scenarios for workflow validation
- Support for multiple state types (prompt, input, workflow_ref)
- Dynamic state routing with LLM-driven decisions
- Error handling with fallback states
- Variable interpolation in workflows
- Custom tools support
- Comprehensive documentation and examples
